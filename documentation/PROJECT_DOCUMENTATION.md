## 3. Abstract

This project demonstrates the development of a responsive static portfolio website and its deployment using Amazon S3 Static Website Hosting. The website presents intern information, skills, project details and contact information.

## 4. Objectives

- Create a professional responsive static website.
- Organize source code into proper folders.
- Use HTML, CSS and JavaScript with clear comments.
- Upload website files to Amazon S3.
- Enable static website hosting.
- Test the public website endpoint.
- Maintain documentation and screenshots.

## 5. Technologies

- HTML5 for structure
- CSS3 for presentation and responsive design
- JavaScript for mobile navigation
- Amazon S3 for cloud hosting
- GitHub for source-code submission

## 6. Implementation

The main file is `index.html`. It loads CSS from `assets/css/style.css` and JavaScript from `assets/js/script.js`. The website is tested locally before being uploaded to Amazon S3. During S3 upload, the same folder structure must be preserved.

## 7. Expected Result

A responsive portfolio website that displays correctly on desktop and mobile devices and can be accessed through an Amazon S3 static website endpoint after deployment.

## 8. Security

AWS credentials and sensitive account information must never be committed to GitHub. Only the minimum required permissions should be used.
