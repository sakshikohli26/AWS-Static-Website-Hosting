# VIVA QUESTIONS AND ANSWERS

1. **What is Amazon S3?**  
   Amazon S3 is an object storage service provided by AWS.

2. **What is a static website?**  
   A website made from client-side files such as HTML, CSS and JavaScript.

3. **Why is S3 suitable for this project?**  
   The project does not require server-side processing or a database.

4. **What is the main file of this project?**  
   `index.html`.

5. **Where is the CSS file stored?**  
   `assets/css/style.css`.

6. **How is CSS linked to HTML?**  
   Using `<link rel="stylesheet" type="text/css" href="./assets/css/style.css">`.

7. **What happens if the folder structure changes?**  
   Relative file paths can break and CSS or JavaScript may not load.

8. **What is responsive design?**  
   A design that adapts to different screen sizes.

9. **What is an S3 bucket?**  
   A container used to store objects in Amazon S3.

10. **What sensitive information should never be uploaded to GitHub?**  
    Access keys, secret keys, passwords, account IDs and billing information.
