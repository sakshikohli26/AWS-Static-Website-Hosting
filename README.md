## How to Run Locally

1. Download the complete ZIP file.
2. Extract the ZIP file.
3. Open the extracted project folder.
4. Double-click `index.html`.

The CSS is linked using:

```html
<link rel="stylesheet" type="text/css" href="./assets/css/style.css">
```

## AWS S3 Deployment

Upload the project while preserving the folder structure:

- `index.html`
- `assets/css/style.css`
- `assets/js/script.js`

If the `assets` folder structure is changed, the CSS and JavaScript links will stop working.

## Security Note

Never upload AWS access keys, secret keys, passwords, account IDs or billing information to GitHub.
